# Biodata Profile

A beautifully designed, responsive Flutter application to display a personal biodata profile.

## Features
- **Clean Interface:** Modern, card-based UI that presents information clearly.
- **Responsive Design:** Automatically adapts to mobile, tablet, and desktop viewports.
- **Categorized Information:** Neatly organizes personal, educational, family, and contact details.

## Tech Stack
- Flutter
- Material 3 Design System

## Getting Started

To run the application:

1. Ensure you have Flutter installed.
2. Clone this repository.
3. Run `flutter pub get` to fetch dependencies.
4. Run `flutter run` to launch the app on your preferred device.
